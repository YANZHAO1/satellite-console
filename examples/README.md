# 示例

本目录包含 Satellite Console 的使用示例，帮助你快速了解和测试各项功能。

## 📋 示例列表

### 1. basic.html - 基础示例

演示 Satellite Console 的基本功能，包括：
- 启动卫星窗口
- 生成不同级别的日志（log、warn、error）
- 测试复杂对象和数组的序列化
- 批量日志生成和性能测试
- 特殊情况处理（循环引用、深层嵌套、长字符串）
- 窗口控制（打开、关闭、状态检查）

**适用场景**：单页面应用调试、功能测试

### 2. multi-page/ - 多页面示例

演示多页面应用场景下的日志聚合功能，包含三个独立页面：
- **page1.html** - 用户管理页面
- **page2.html** - 订单管理页面
- **page3.html** - 数据分析页面

每个页面模拟不同的业务功能，所有日志统一显示在卫星窗口中。

**适用场景**：多页面应用（MPA）、微前端应用调试

## 🚀 使用方法

### 前置条件

1. 确保已安装依赖：
```bash
npm install
```

2. 构建项目：
```bash
npm run build
```

### 运行基础示例

1. 在浏览器中打开 `examples/basic.html`
2. 卫星窗口会自动打开
3. 点击页面上的按钮测试各种功能
4. 在卫星窗口中查看日志输出

### 运行多页面示例

1. 在浏览器中打开 `examples/multi-page/page1.html`
2. 卫星窗口会自动打开
3. 点击页面上的链接打开其他页面（page2.html、page3.html）
4. 在不同页面中执行操作，观察卫星窗口中的日志聚合效果
5. 使用卫星窗口的来源筛选器过滤特定页面的日志

## 💡 测试建议

### 基础功能测试

1. **日志级别区分**：点击"生成基础日志"，观察不同级别日志的颜色区分
2. **复杂对象展示**：点击"测试复杂对象"，查看对象和数组的展开/折叠功能
3. **性能测试**：点击"生成 1000 条日志"，测试虚拟滚动和渲染性能
4. **特殊情况**：测试循环引用、深层嵌套等边界情况的处理

### 多页面场景测试

1. **日志聚合**：在多个页面中同时产生日志，验证卫星窗口能接收所有日志
2. **来源筛选**：使用来源筛选器，只查看特定页面的日志
3. **搜索功能**：在搜索框中输入关键词（如"[页面1]"），过滤特定内容
4. **窗口管理**：关闭卫星窗口后，在任意页面重新打开，验证日志继续接收

### 错误处理测试

1. **弹窗拦截**：如果浏览器阻止弹窗，手动允许后重试
2. **窗口关闭**：关闭卫星窗口，验证业务页面继续正常工作
3. **异常日志**：测试错误日志的堆栈信息显示

## 🔧 故障排除

### 卫星窗口没有打开

- 检查浏览器是否阻止了弹出窗口
- 在浏览器地址栏右侧查看弹窗拦截图标，点击允许
- 或者手动调用 `SatelliteConsole.launch()` 重新打开

### 日志没有显示

- 确认卫星窗口已打开（使用 `SatelliteConsole.isWindowOpen()` 检查）
- 检查浏览器控制台是否有错误信息
- 确认浏览器支持 BroadcastChannel API（Chrome 54+、Firefox 38+、Safari 15.4+）

### 多页面示例中日志混乱

- 使用卫星窗口的来源筛选器，按页面过滤日志
- 使用搜索功能，输入页面标识（如"[页面1]"）过滤
- 点击清空按钮，清除历史日志后重新测试
